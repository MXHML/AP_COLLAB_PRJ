Link JavaScript to HTML

Well how do I use JavaScript on a webpage?

It's simple!

JavaScript can be used on a HTML page in two ways:


The <script> tag

The <script> tag is the easiest way to use JavaScript inside an HTML page, all you have to do is put the JavaScript code inside the <script> tag and it will run when the page loads.

Here is an example:

<script>
alert("Hello World!");
</script>

When this the page loads, the browser will show an alert containing 'Hello World!'

External JavaScript

External JavaScript is a way to use JavaScript on a HTML page without using the <script> tag.

Here is an example:

<script src="script.js"></script>

When the page loads, the browser will look for a file called 'script.js' and run the code inside it.

(Typically, external JavaScript files are loaded AFTER the body tag. The order you load them matters!)
