JavaScript Functions

What are JavaScript functions? How do I use them?

JavaScript functions are the backbone of a JavaScript program

How do they work?

Functions are a way to group code together to perform a specific task. They are reusable and can be called as many times as you want. They are also a way to make your code more readable and easier to understand.

Here is an example of a function:

function myFunction() {
document.getElementById("demo").innerHTML = "Hello World!";
}

You can also pass parameters to functions too!

Here is an example of a function with parameters:

function myFunction(p1, p2) {
return p1 * p2;
}

Here is an example of how to call the function with parameters:

var x = myFunction(4, 3);
