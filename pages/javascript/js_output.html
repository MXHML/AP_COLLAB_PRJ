How do I read/display data from JavaScript?

JavaScript can display data in a few ways:


  • Writing into the HTML output using document.write()
    Using this will delete ALL HTML from your page
  • Writing into an alert box, using window.alert()
  • Writing into the browser console, using console.log()
  • Writing into an HTML element, using the element.innerHTML property
