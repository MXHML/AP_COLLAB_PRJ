JavaScript Variables

How do I declare and use variables in JavaScript?

Declaring variables in JavaScript is simple:


  • Use the var keyword to declare a variable.
  • Use the let keyword to declare a variable
  • Use the = operator to change the value of a variable.

Example:

var x = 5;

let y = 6;

var z = x + y;

z = document.getElementById('example');

JavaScript also has arithmatic operators

Arithmatic Operators

Arithmatic operators are used to perform mathematical operations on numbers.

JavaScript has the following arithmatic operators:

  • + - Addition
  • - - Subtraction
  • * - Multiplication
  • / - Division
  • % - Modulus
  • ++ - Increment
  • -- - Decrement

Addition Example:

var x = 5;

var y = 6;

var z = x + y;

z=11

Subtraction Example:

var x = 5;

var y = 6;

var z = x - y;

z=-1

Multiplication Example:

var x = 5;

var y = 6;

var z = x * y;

z=30

Division Example:

var x = 5;

var y = 6;

var z = x / y;

z=0.8333333333333334

Modulus Example:

var x = 5;

var y = 6;

var z = x % y;

z=5
