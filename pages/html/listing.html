HTML Listing

HTML allows user to list items orderly or unorderly

There are three types of lists in HTML

Ordered List

Unordered List

Ordered List

Ordered list is a list where each item is numbered

Unordered List

Unordered list is a list where each item is marked with a bullet

Description List

Description list is a list where each item has a description

Example

Ordered List

<ol style=list-style-position:inside;"> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ol>
  1. Item 1
  2. Item 2
  3. Item 3

Unordered List

<ul style="list-style-position:inside;"> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul>
  • Item 1
  • Item 2
  • Item 3

Description List

<dl< <dt<Item 1</dt< <dd<Item 1 Description</dd< </dl<
