HTML is structured with elements which is defined by tags

Defines the root of an HTML document

Defines the head of an HTML document

<title>

Defines the title of an HTML document

<body>

Defines the body of an HTML document

<h1> to <h6>

Defines HTML headings

<p>

Defines a paragraph

<br>

Inserts a single line break

<hr>

Defines a thematic break in an HTML page
