Welcome to HTML!

HTML elements can be given a class to alter multiple styles of the same category item at once

Classes are defined in the head of the HTML document

Classes are defined with a period before the name of the class

Example

<h1 class="red">This is red</h1>
<h1 class="blue">This is blue</h1>
<h1 class="green">This is green</h1>
