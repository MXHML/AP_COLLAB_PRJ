Welcome to HTML!

HTML can be styled without a CSS page with the style tag

HTML Styles

<tag style="property:value;">

HTML Styling Syntax

Example:

<p style="color:blue;">This is a paragraph.</p>

Output:

This is a paragraph.

Properties:

color

background-color

font-size

font-family

text-align
